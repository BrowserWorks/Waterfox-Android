Waterfox for Android は、ユーザーがコントロールできるプライバシー重視のモバイル ブラウザーです。完全にオープン ソースで作成された Waterfox は、分析、テレメトリ、不要なクローズド ソース統合を取り除き、代わりにクリーンなプライベート ブラウジング エクスペリエンスを提供します。

他の Android Web ブラウザとは異なり、Waterfox には使用状況を追跡する分析やテレメトリがありません。また、クローズド ソース SDK も使用せず、デフォルトで有効になっているスポンサー付きショートカットもありません。クローズド ソースの Pocket read-it-later サービスなどの統合は削除されました。さらに、中国の検閲に関連するコードはすべて削除されました。 

ウォーターフォックスはプライバシーを最優先に考えています。オンライントラッカーをブロックし、企業があなたの閲覧習慣をプロファイリングするのを防ぎます。検閲を防ぐために、Oblivious HTTP 上の DNS を使用した暗号化 DNS サポートが付属しています。このユニークな機能により、DNS クエリはネットワーク操作を避けるために隠され、プレミアム機能を無料で提供する唯一の Web ブラウザーです。

カスタマイズに関しては、Waterfox はアドオンをサポートするため、モバイル ブラウジング エクスペリエンスを強化できます。履歴、Cookie、ズーム レベルなどのカスタマイズ可能な設定もあります。あなたは完全にコントロールできます。

Waterfox は内部で、最大のオープン ソース Web ブラウザを動かしているのと同じ超高速の Gecko エンジンを使用しています。ページはすばやく読み込まれ、Web サイトはスムーズに動作します。Waterfox は、最高のブラウジング体験を実現する最新の Web 標準とテクノロジーもサポートしています。

Waterfox は、ユーザーのプライバシーと選択に重点を置いて独自に開発されました。バックグラウンドでメトリクスやデータ収集が行われることはありません。Waterfox for Android は、デスクトップ クラスのプライバシーをモバイルにもたらします。

Waterfox を試してみて、モバイル ブラウジングを制御してください。Web をプライベートにブラウズできます。
